Project Overview
This project demonstrates an AI-powered website copy generator built using prompt engineering techniques.
The system generates conversion-focused website content for local businesses, specifically demonstrated using a Fitness Gym use case.
The goal of this task is to design structured prompts that produce clear, persuasive, and website-ready copy, similar to real client projects.
Business Use Case
Business Type: Fitness Gym
Target Audience: Working professionals and students
Tone: Motivational, friendly, professional
Objective: Increase gym memberships through effective website copy
Tools Used
ChatGPT (for prompt execution and content generation)
GitHub (for documentation and version control)
Markdown (for structured content files)

 Prompt Engineering Approach
The project uses a modular prompt system, where each prompt has a specific role:
Business Context Prompt: Defines audience, tone, and unique selling points
Homepage Prompt: Generates structured homepage content
Services Prompt: Creates service descriptions based on offerings
CTA Prompt: Produces strong call-to-action sections
This approach ensures consistency, reusability, and business relevance.
 Key Features
Homepage copy with clear value proposition
Service descriptions tailored for a fitness gym
Conversion-focused call-to-action sections
Simple and professional language suitable for real websites
